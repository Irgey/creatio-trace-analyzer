Creatio Trace Analyzer

Browser-based utility for analyzing Creatio trace JSON files. It helps inspect slow events, SQL queries, parameters, stack traces, event data, and execution timelines.

Data is processed locally in the browser. No backend is required.

Endpoint to run tracing: 0/api/Tracing/v1/Index (i.e. https://site.creatio.com/0/api/Tracing/v1/Index)

Repository: github.com/Irgey/creatio-trace-analyzer
